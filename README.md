# poker_agent




Poker Agent is an AI-driven intelligent agent system for poker games, designed to achieve high-level poker decision-making capabilities through machine learning and agent-based reasoning.



\## Status / Roadmap



🚧 **Work in progress.**  

This repository is under active development. The codebase will be **open-sourced progressively** (modules will be released in stages) as components are stabilized, documented, and tested.



Planned releases may include (subject to change):

\- Core agent framework (state, action, logging, interfaces)

\- Poker environment adapters & hand evaluation utilities

\- Decision modules (baseline policy, exploit layer, strategy selection)

\- Training & evaluation scripts

\- Demo UI / API service
